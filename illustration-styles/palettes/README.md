# Palette Details

Each palette in `references/palettes.md` has a one-line description sufficient for prompt composition. Per-palette detail pages (`palettes/<id>.md`) exist for the most-used palettes only; see `references/palettes.md` for the full list and `references/index.json` for machine-readable metadata.

To add a per-palette detail page, follow the template at `palettes/_template.md`.
