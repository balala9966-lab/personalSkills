# Macaron

Soft macaron tones on warm cream. The default educational palette.

## Colors

| Role | Hex | Notes |
|------|-----|-------|
| Primary blue | `#A3C9E1` | Cool soft accent |
| Mint | `#A8DCC6` | Secondary accent |
| Lavender | `#C8B4E1` | Tertiary accent |
| Peach | `#F4C6A6` | Warm accent |
| Background | `#FAF3E8` | Warm cream base |
| Ink | `#3C3C3C` | Soft dark for line work, never pure black |

## Use Case

Educational content, knowledge-sharing pieces, tutorial articles. The warm cream base + soft accents create a "gentle teacher" register.

## Pairs Well With

- Styles: `notion`, `sketch-notes`, `vector-illustration`, `flat`
- Types: `infographic`, `flowchart`, `framework`, `comparison`

## Anti-Pairs

- `blueprint`, `scientific` (cool palette conflicts)
- `screen-print` (poster boldness overpowers the softness)
- Dark / high-contrast styles

## Generation Prompt Hints

Include: "soft macaron palette: cool blue, mint, lavender, peach, on warm cream base, soft dark ink instead of pure black, educational gentle register"
