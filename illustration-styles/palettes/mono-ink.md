# Mono Ink

Pure white background, black ink, sparse semantic accent colors. The canonical palette for visual essays and `ink-notes` style.

## Colors

| Role | Hex | Notes |
|------|-----|-------|
| Ink | `#000000` | Primary line work and main labels |
| Background | `#FFFFFF` | Pure white, no texture |
| Accent red | `#D9483B` | Reserved for problems / outcomes / emphasis |
| Accent orange | `#E47A2A` | Reserved for flow / movement / arrows |
| Accent blue | `#3A6FB0` | Reserved for system state / secondary notes |

## Use Case

Visual essays, manifestos, "anatomy of X" pieces, conceptual diagrams where the writer wants restraint to do the visual work.

## Pairs Well With

- Styles: `ink-notes` (canonical), `vector-illustration`, `editorial`, `minimal`
- Types: `infographic`, `comparison`, `framework`

## Anti-Pairs

- `warm`, `watercolor`, `playful` — accent discipline gets diluted
- `pixel-art`, `vintage` — too much visual character

## Generation Prompt Hints

Include: "pure white background, black hand-drawn line work, sparse semantic accents — red for problems/outcomes, orange for flow/movement, blue for system state — color use restrained, less is better"
