# Screen Print

Bold poster art with halftone textures and limited color. The op-ed cover register — high visual impact for opinion and culture.

## Visual Traits

- **Line work**: bold, sometimes silhouetted, with stencil-cut edges
- **Fill**: flat 2-5 color blocks
- **Texture**: visible halftone dot pattern
- **Background**: solid color or simple gradient
- **Color discipline**: limited palette (2-5 colors), used for symbolic effect, not realism
- **Typography**: bold display type when present
- **Composition**: symbolic figures, negative space telling a secondary story, retro-poster aesthetic

## Best For

- `scene` — when the scene needs to *mean* something, not depict something
- `comparison` — split duotone compositions
- `framework` — geometric node hierarchies with poster aesthetics
- Cultural commentary, op-eds, manifestos, cinema-style dramatic narrative

## Avoid For

- `flowchart` — the bold aesthetic fights step-by-step clarity
- `infographic` — better tools exist for clean data display
- Lifestyle, soft narrative — too aggressive

## Canonical Palette

Pairs best with: `duotone` (two-color tension is the whole point) or `vivid`. Also works with `mono` for stark editorial pieces.

## Generation Prompt Hints

- "soft watercolor", "warm hand-drawn"
