# Scientific

Academic precision, lab-figure feel — the look of a Nature paper figure.

## Visual Traits
- Precise technical line work
- Axis labels, measurement marks, data points
- Restrained palette (often grayscale + 1-2 data colors)
- Lab-figure typography (often serif or technical sans)

## Best For
`infographic`, `framework` — research findings, scientific explainers, paper-grade content.

## Avoid For
`scene`, `flowchart` (better technical options exist), narrative content.

## Generation Prompt Hints
Include: "scientific figure style, precise data visualization, axis labels, restrained palette, academic paper aesthetic"
Avoid: "decorative", "playful", "warm"
