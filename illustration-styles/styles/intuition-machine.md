# Intuition Machine

Technical briefing on aged paper — the "engineering field notebook" aesthetic, technical-precise but warmly aged.

## Visual Traits
- Technical-precise line work
- Aged paper texture or pale cream background
- Restrained palette with one accent
- Notebook / briefing-page layout

## Best For
`infographic`, `flowchart`, `framework` — academic-style briefings, intuition-building explainers.

## Avoid For
Bright modern contexts.

## Generation Prompt Hints
Include: "technical briefing on aged paper, engineering notebook style, precise lines with warm texture"
Avoid: "neon", "polished vector", "modern flat"
