# Blueprint

Technical schematic / engineering blueprint feel. For when the article reads like a spec.

## Visual Traits

- **Line work**: precise thin lines, often with grid underlay
- **Fill**: none or single tint
- **Background**: clean white or very pale blue
- **Color discipline**: blue dominant, white grid, occasional red highlight for callouts
- **Typography**: monospace or technical sans, label-heavy
- **Composition**: grid-aligned, clear zones, technical annotations

## Best For

- `infographic`, `flowchart`, `framework` — anywhere precision matters
- System design, architecture, API anatomy, technical breakdowns

## Avoid For

- `scene`, `timeline` — the schematic feel suppresses narrative
- Lifestyle, emotional content — too sterile

## Canonical Palette

Pairs best with: `cool` for tech-forward feel, default blue-on-white for engineering classic. Avoid `warm` (color clash).

## Generation Prompt Hints

Include:
- "technical blueprint / schematic style"
- "precise thin lines, grid layout"
- "blue and white color scheme"
- "technical annotations, monospace labels"
- "engineering diagram aesthetic"

Avoid:
- "hand-drawn", "wobble", "organic" — kills precision
- "warm colors", "painterly"
